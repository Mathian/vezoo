Соглашение оператора

1. Роль оператора

Оператор осуществляет приём и обработку заказов для одного заведения-партнёра Vezoo. Доступ предоставляется администратором заведения.

2. Обязанности

Оператор обязан своевременно принимать входящие заказы, корректно передавать их на кухню и организовывать доставку через курьеров.

3. Один оператор — одно заведение

Оператор может работать только с одним заведением одновременно. При добавлении к новому заведению требуется отвязаться от текущего.

4. Конфиденциальность

Данные клиентов (ФИО, телефон, адрес) используются только для обработки и доставки заказов. Передача третьим лицам запрещена.
